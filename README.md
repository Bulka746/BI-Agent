Для запуска агента необходимо указать свой API-ключ в скрипте test.py и запустить скрипт app(3).py при запущенном контейнере dalakehouse_bi_agent.

Для запуска полного агента вместе с фронтендом используется app(3).py. Запрос вводится через меню приложения.

Для просмотра и анализа кода рекомендуется использовать файл test.ipynb (именно формат ipynb), так как в нём отображаются логи, код представлен в более читаемом виде, а также после выполнения можно просмотреть значения переменных состояния.
